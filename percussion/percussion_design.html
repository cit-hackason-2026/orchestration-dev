0. 音響基礎と担当楽器の特性調査（設計前のリサーチ）

なぜ先にこれをやる？
  • 音色設計は「物理的に何がどうなっているか」を理解していないと、パラメータをいじっても勘頼みになる。
  • フーリエ変換 → 非調和モード構造 → 実測スペクトログラム の順で「目標の音が物理的にどんな構成か」をつかんでから設計に入ると、根拠を持って数値を決められる。
  • パーカッションは旋律楽器と違って音が一瞬で終わる。だから普通の FFT（時間平均）だけでなく、スペクトログラム（時間 × 周波数） を見ることが特に重要。
  • ここで得た知見は 2 章（音源生成方式）・5 章（パーカッション特有の検討） で直接使う。

「任意の波形は正弦波の和として表せる（フーリエ級数）」／「時間波形 ↔ 周波数スペクトルの変換」／「FFT は離散版の高速アルゴリズム」の 3 点を 3〜5 行で説明できるように。
調べ先：Wikipedia「フーリエ変換」「高速フーリエ変換」、YouTube の直感的解説動画（3Blue1Brown など）も有効。

基音（fundamental）・整数倍音（harmonics）・非調和部分音（inharmonic partials, modes）の違い。
クラリネットやフルートのような気柱振動／弦振動は周波数比が整数倍に近い → 「ピッチを感じる音」。
一方、ドラム（膜振動）・シンバル（板振動）・木魚（棒振動）は固有モードが整数比にならない → 「ピッチ感が薄い／ない音」。
例：理想的な丸い膜の振動モード比は 1 : 1.59 : 2.14 : 2.30 : 2.65 …（ベッセル関数の零点）で、整数比からズレている。これが「ドンッ／カンッ」の打楽器らしさの正体。

打楽器は振動体の種類で 3 系統に分けて理解すると整理しやすい：

  • 膜（メンブラフォン）：キックドラム、スネア、タム、ティンパニ。打面が面振動 → 低めの基音 + 多数の非調和モード。スネアはさらに裏面のスナッピー（金属ワイヤ）でノイズ成分が乗る。
  • 板／棒（イディオフォン）：シンバル、トライアングル、ウッドブロック、シロフォン。金属系はモード密度が高くノイズに近い、木系はモードが少なく明瞭なピッチ感が出やすい。
  • 電子的にエミュレートされたもの：TR-808 / 909 のキックは「サイン波 + ピッチエンベロープ」、スネアは「サイン 2 本 + バンドパスノイズ」と、物理現象を抽象化した合成で作られる。今回は実装容易性からこちら寄りで設計するのが現実的。
調べ先：Wikipedia「打楽器」「メンブラノフォン」「TR-808」、Sound on Sound 誌の "Synth Secrets" シリーズ（特に Drum Synthesis 回）。

YouTube / freesound.org / drum sample pack などから、採用するドラム種別ごとに単音（ワンショット）を確保。理想は「アコースティックドラム」と「TR-808/909 系」を両方並べて聞き比べると、合成の方向性が決めやすい。使った音源名・URL を下に記録。

  1. 音源を Audacity で開く

💡 コツと注意点（調べ方）
  • ピッチ成分の「下降スイープ」を確認：キック・タムは打った瞬間が最も高く、すぐに低くなる（ピッチエンベロープ）。スペクトログラムで斜めに下がる線として見える。これが合成の鍵。
  • ノイズ成分と調和成分の比率：ハイハット・シンバルはほぼ全帯域ノイズ、スネアは「サイン的成分 + ノイズ」の混合、キックはほぼ調和（サイン的）成分。
  • 減衰時間 T60：音が初期から 60 dB 落ちるまでの時間。キックは 200〜400ms、スネアは 100〜200ms、ハイハットは 50〜150ms（クローズド）/ 数秒（オープン）など、ドラム種別で全然違う。
  • サンプル音源は WAV/FLAC を優先：MP3 は高域のシャーシャー感（シンバル系で重要）が劣化することがある。

1. パーカッション音色の表現方針

打楽器は「Attack が極端に短い／ほぼ 0」のが特徴。Attack を 0ms にするとクリックノイズが出るので、1〜5 ms 程度の極短 Attack で立ち上げる、あるいは波形の頭にクリックを意図的に足すなどの整理。

キック／タム＝ピッチ感あり、ハイハット／シンバル／スネア＝ピッチ感薄め、という分類を踏まえ、採用するドラムごとに「ピッチ成分とノイズ成分の比」を決める。

バンドパスノイズの帯域をドラムごとに変える、初期のクリック成分を足す、わずかなリバーブを掛ける、など。シンプル実装で済ませるなら「胴鳴り＝主モード周波数の短いサイン波」で十分。

きらきら星の伴奏なら最低でも「低音（キック）／中音（スネアまたは木片系）／高音（ハイハット）」の 3 種類は欲しい。各ドラムでベース構成・ノイズ帯域・減衰時間が明確に違うこと。

2. 音源生成方式

定番の合成方式：

  • キック：サイン波 + 急降下するピッチエンベロープ（例：150Hz→50Hz を 30ms で）+ ごく短いクリック
  • スネア：サイン 2 本（180Hz, 330Hz 程度）+ バンドパスノイズ（中心 1〜2kHz）
  • ハイハット（クローズド）：ホワイトノイズ + HPF（カットオフ 8〜10kHz）+ 短い AD
  • ウッドブロック／クラベス：高めのサイン波 2 本（800Hz, 2400Hz 程度）+ 極短減衰
どの方式を採用するかと、その根拠を書く。

ホワイト／ピンク／バンドパスのどれを使うか。ハイハットは HPF 8kHz 以上、スネアは BPF 1〜3kHz など、ドラムごとに帯域が違う。

「初期ピッチ → 持続ピッチ」「下降に要する時間」を決める。キックは 150→50Hz を 30〜50ms で下降が定番。下降が速すぎると「ピッ」、遅すぎると「ボーン」になる。

ノイズ整形のための BPF / HPF が中心。キック・タムは LPF をかけて鋭さを抑えるのも定番。

パーカッションは Sustain なしの AD（または AHR）でよい。Attack 1〜5ms、Decay = T60 を基準に決める。Release は基本不要（鍵盤を「離す」概念がない）。

きらきら星の伴奏として何種類用意するか。最低 2 種類（キック・ハイハットなど）、推奨 3 種類（キック・スネア・ハイハット）。

3. Arduinoから送るデータとの対応

旋律楽器とは違い、NOTE 番号（音程）はほぼ不要。代わりに：

  • 楽器種別（パーカッション）
  • ドラム種別（キック=0 / スネア=1 / ハイハット=2 …）
  • 発音タイミング（ヒット）
  • 強弱（ベロシティ）：必要なら入れる。リズム楽器でアクセントを付けたい場合に有効
どの項目が必要十分か判断。

フルート・クラリネット・オルガンと同じ 5〜7 バイトのバイナリパケットに乗せる。NOTE フィールドを「ドラム種別 ID」として流用するのが楽（パーカッションでは MIDI ノート番号は不要なので）。3 人と統一仕様を確認。

パーカッションは輪唱に参加せず、SYNC（小節頭マーカー）を直接受け取って独自にリズムパターンを発火する。SYNC 受信 → 当該小節のリズムパターン配列を参照 → 各拍／半拍／16 分音符のタイミングで Processing にヒットイベントを送出、という流れ。

4. Processing側の再生処理

受信 → パース → ドラム種別で分岐 → 該当 Voice をトリガ。1 回のヒットは数百 ms で終わるので、Note Off の概念は不要（または velocity=0 を「ハイハットのオープン → クローズへの切り替え」用に流用してもよい）。

楽器種別 = パーカッション だったときに、ドラム種別 ID を見て kick.hit() / snare.hit() / hihat.hit() のどれを呼ぶか分岐する。

キックとハイハットは同時に鳴ることがある。各ドラムを独立した Voice として持ち、互いにブロックしないこと。同じドラムが連打された場合は前の音をフェードアウトしてから再トリガ。

Attack を完全に 0ms にしないこと（最低 1ms 推奨）。前のヒットがまだ鳴っている状態で再トリガする場合は、波形の急な不連続を避けるためクロスフェード（数 ms）する。

パーカッションは数 ms のズレが致命的に聞こえる。シリアル受信から発音までの遅延を最小化する。Processing 側で Serial を `bufferUntil` でなく `available()` ループでタイトに読む、ノイズフロアの抑制、など。

5. パーカッション特有の検討事項

「整数倍音」ではなく「モード周波数（非整数比）」で組む。スネアならサイン 2 本を 180Hz と 330Hz など整数比でない周波数で重ねる。

「サイン + ピッチエンベロープ」（キック）と「フィルタ済みノイズ」（ハイハット）は実装が全く別物。両方の Voice クラスを別々に用意する。

指揮者からの SYNC（global_bar / BPM）に直接同期し、entry_offset = 0 で第 0 小節から第 12 小節まで全部鳴らす。ループパターン（4/4 で 1 小節分）を保持して、SYNC ごとにそれを発火するのが最小実装。

時間がないときに何だけ残せば「ドラムっぽく」聞こえるか。例：キック（サイン + ピッチ降下）+ ハイハット（ノイズ + HPF + 短い減衰）の 2 種類だけでも、4 つ打ち＋裏拍ハットでバンド感は出る。
